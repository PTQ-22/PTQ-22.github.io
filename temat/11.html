Liczby losowe i XOR

XOR - różnica symetryczna bitowa. (w c++ operator ^)

Zwraca prawdę tylko, gdy bity są różne.

0 i 0 => 0

0 i 1 => 1

1 i 1 => 0

1 i 0 => 1

Szyfrowanie z XOR polega na ustawieniu zarodka generatora liczb pseudolosowych na wartość klucza, generowaniu losowej liczby dla każdego szyfrowanego znaku i wykonaniu na niej operacji XOR z obecnym znakiem. Jest samoodwracalne.

string random_xor(string text, int key)
{
    srand(key);
    string encrypted = "";

    for (char c : text) {
        int x = rand() % 64;
        encrypted += (c ^ x);
    }

    return encrypted;
}
