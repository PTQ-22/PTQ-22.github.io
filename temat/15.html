Stos

Stos - dynamiczna struktura danych, w której mamy dostęp tylko do elementu znajdującego się na wierzchołku stosu. Możemy dodawać elementy na wierzchołek stosu oraz zdejmować elementy z wierzchołka. LIFO - last in, first out.

struct Element {
    int value;
    Element* under;
};

Element* stack = nullptr;

void push(int x) 
{
    Element* tmp = new Element;
    tmp->value = x;
    tmp->under = stack;
    stack = tmp;
}

int pop() 
{
    int res = stack->value;
    // zapamiętujemy obecny wierzchołek
    Element* tmp = stack; 
    stack = tmp->under;
    delete tmp; // usuwamy stary wierzchołek
    return res;
}
